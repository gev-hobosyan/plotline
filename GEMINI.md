# Project Overview

This is a Next.js project bootstrapped with `create-next-app`. It appears to be a web application for tracking movies and books. The project uses Next.js for the frontend and backend, with TypeScript as the primary language. It uses `drizzle-orm` for the database and `kinde-auth` for authentication.

# Building and Running

To get started with the development server, run the following command:

```bash
npm run dev
```

This will start the development server on [http://localhost:3000](http://localhost:3000).

Other useful commands:

*   `npm run build`: Builds the application for production.
*   `npm run start`: Starts a production server.
*   `npm run db:generate`: Generates database migration files.
*   `npm run db:migrate`: Applies database migrations.

# Development Conventions

The project uses TypeScript and follows the standard Next.js project structure. It uses CSS modules for styling, as indicated by the `globals.css` and `*.css` files. The project also uses `tailwind-css` for styling.
